Chapter 1. What is Bootique

Bootique is a minimally opinionated technology for building container-less runnable Java applications. No JEE container required! It is an ideal platform for microservices, as it allows you to create a fully functional app with minimal-to-no setup. It is not limited to a specific kind of app (or the "micro" size) though and can be used for REST services, webapps, runnable jobs, DB migrations, JavaFX GUI apps to mention a few examples.

Bootique was inspired by two similar products - Dropwizard and SpringBoot, however its focus is different. Bootique favors modularity and clean pluggable architecture. Bootique is built on top of Google Guice DI container, which provides the core of its modularity mechanism.
